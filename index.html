ßetamax by robfletcher

A Groovy record/playback proxy for testing applications that access external HTTP resources. Inspired by Ruby's VCR.

Dependencies

  • Groovy
  • Apache HttpClient
  • Apache HttpCore NIO Extensions
  • SnakeYAML
  • JUnit 4
  • Apache log4j
